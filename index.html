Pyenv

Pyenv é a ferramenta que utilizo para gerenciar múltiplas versões do interpretador Python. É uma ferramenta simples de instalar e configurar (basta seguir as informações do repositório no github).

Pyenv permite instalar versões diferentes do Python (como python2, python3, pypy, entre outras) em um mesmo ambiente de desenvolvimento. Uma das grandes vantagens na minha opinião, é poder usar versões menores do interpretador Python sem conflito. Por exemplo, no Linux Mint 19, a versão instalada do Python3 no sistema é a 3.6.5. Utilizando Pyenv, é possível utilizar a versão 3.7.0 do interpretador, junto com a versão do sistema, sem qualquer tipo de conflito.

É fundamental incluir as dependências descritas em Common Build Problems.
